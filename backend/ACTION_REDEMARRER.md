# ⚠️ Action Requise : Redémarrer le Backend

Les colonnes "Responsable" apparaissent vides car le serveur (Backend) utilise encore l'ancienne version du code en mémoire.

## Procédure

1. Allez dans le terminal qui exécute le Backend (API).
2. Arrêtez le serveur (Combinaison `Ctrl + C`).
3. Relancez le serveur (ex: `python run.py` ou `uvicorn app.main:app --reload`).
4. Rechargez la page Web (`F5`).

La colonne **Responsable** devrait alors se remplir avec le nom du poste.
